Full-Stack E-Commerce Platform
🚀 Project Overview
This project is a comprehensive Full-Stack E-Commerce Web Application developed as part of the CodeAlpha Internship task. The goal was to build a functional online store where users can browse products, manage a shopping cart, and complete orders, supported by a robust backend and database.

🛠️ Tech Stack
Frontend: HTML5, CSS3, JavaScript (ES6+)

Backend: Node.js, Express.js

Database: MySQL

Authentication: JWT / Sessions (Login & Signup System)

✨ Key Features
🌐 Frontend (User Interface)
Home Page: Featuring top products and categories.

Product Listings & Details: Dedicated pages for viewing full product information.

Collection Page: Filterable and organized product views.

Shopping Cart: Add/Remove functionality with real-time price calculation.

Checkout System: Secure flow for finalizing orders.

⚙️ Backend & Database
User Authentication: Fully functional Login and Signup system.

Order Management: System to process and store customer orders.

Admin Panel: A private dashboard to manage products, users, and view sales/orders.

Relational Database: MySQL used to store structured data for Products, Users, and Orders.

📂 Project Structure
Plaintext

├── backend/            # Node.js & Express.js server files
├── frontend/           # HTML, CSS, and JS files
├── config/             # Database connection (MySQL)
└── README.md           # Project documentation
🚀 How to Run Locally
Clone the repository: git clone [Your-Repo-Link]

Import the database.sql file into your MySQL Workbench.

Navigate to the backend folder and install dependencies: npm install

Start the server: node server.js

Open index.html in your browser.
