<h1 align="center">✨ Fullstack Chat & Video Calling App ✨</h1>



Highlights:

### 🚀 Key Features

- 🌐 **Real-time Messaging**: Instant delivery with Typing Indicators & Reactions.
- 📹 **Advanced Video Conferencing**: 1-on-1 and Group calls with Screen Sharing & Recording.
- 🎨 **Interactive Collaborative Whiteboard**: Real-time drawing and brainstorming for teams.
- 🌍 **Language Exchange Hub**: Discover language partners for practice and connection.
- 🔐 **Secure Authentication**: JWT-based protected routes and Bcrypt data encryption.
- ⚡ **Modern Tech Stack**: React, Express, MongoDB, TailwindCSS, and TanStack Query.
- 🧠 **Efficient State Management**: Powered by Zustand for a lag-free experience.
- 🎯 **Scalable Architecture**: Integrated with Stream SDK for high-fidelity media.
- 🛡️ **Robust Error Handling**: Comprehensive validation on both Frontend & Backend.
- 🚀 **Cloud Ready**: Optimized for deployment and high scalability.

---

## 🧪 .env Setup

### Backend (`/backend`)

```
PORT=5001
MONGO_URI=your_mongo_uri
STEAM_API_KEY=your_steam_api_key
STEAM_API_SECRET=your_steam_api_secret
JWT_SECRET_KEY=your_jwt_secret
NODE_ENV=development
```

### Frontend (`/frontend`)

```
VITE_STREAM_API_KEY=your_stream_api_key
```

---

## 🔧 Run the Backend

```bash
cd backend
npm install
npm run dev
```

## 💻 Run the Frontend

```bash
cd frontend
npm install
npm run dev
```
